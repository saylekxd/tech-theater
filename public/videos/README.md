# Filmy Animacji

Umieść tutaj 3 filmy animacji postaci:

## Wymagane pliki:

1. **waiting.mp4**
   - Postać czeka na interakcję
   - Film będzie loopowany
   - Zalecany czas: 3-10 sekund

2. **listening.mp4**
   - Postać słucha użytkownika
   - Film będzie loopowany podczas nagrywania i przetwarzania
   - Zalecany czas: 3-10 sekund

3. **responding.mp4**
   - Postać mówi/odpowiada
   - Film będzie loopowany podczas odtwarzania audio odpowiedzi
   - Zalecany czas: 5-10 sekund (najdłuższy)

## Specyfikacja techniczna:

- **Format**: MP4, H.264 codec
- **Rozmiar**: <10MB każdy plik
- **Rozdzielczość**: 1080p lub 720p (zależnie od potrzeb)
- **FPS**: 24-30 fps
- **Kompresja**: Zalecane ustawienia dla web (medium quality)

## Uwagi:

- Filmy będą automatycznie loopowane
- Upewnij się, że ostatnia klatka płynnie przechodzi w pierwszą (seamless loop)
- Dla najlepszego efektu, używaj podobnego oświetlenia i kompozycji we wszystkich 3 filmach
- Testuj na różnych rozdzielczościach ekranu
